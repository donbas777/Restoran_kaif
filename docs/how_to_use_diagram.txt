# Як використати діаграму бази даних

1. Відвідайте сайт https://dbdiagram.io/

2. Натисніть кнопку "Create your diagram" або "New Diagram"

3. Видаліть приклад коду, який відображається за замовчуванням

4. Скопіюйте та вставте вміст файлу database_diagram.dbml у редактор

5. Діаграма буде автоматично згенерована та відображена у правій частині екрану

6. Ви можете:
   - Експортувати діаграму як зображення (PNG, PDF)
   - Зберегти діаграму для подальшого редагування
   - Поділитися посиланням на діаграму з іншими

7. Для редагування діаграми:
   - Змінюйте код у лівій панелі
   - Перетягуйте таблиці для зміни їх розташування
   - Використовуйте інструменти форматування для налаштування вигляду

8. Для експорту в SQL:
   - Натисніть кнопку "Export"
   - Виберіть потрібний формат (PostgreSQL, MySQL, SQL Server тощо)
   - Завантажте згенерований SQL-скрипт
