# Опис фронтенду ресторану "La Riviera"

## Технології та бібліотеки

- **HTML5/CSS3** - основа для розмітки та стилізації сторінок
- **Bootstrap 5** - фреймворк для створення адаптивного та мобільного дизайну
- **JavaScript/jQuery** - для інтерактивності та динамічних елементів
- **Font Awesome** - бібліотека іконок
- **AOS (Animate On Scroll)** - бібліотека для анімацій при прокрутці
- **Google Fonts** - веб-шрифти (Playfair Display, Poppins)

## Структура шаблонів

### Базовий шаблон (base.html)
- Містить загальну структуру сторінок (head, header, footer)
- Підключає всі необхідні стилі та скрипти
- Містить навігаційне меню та футер

### Головні сторінки
- **home.html** - головна сторінка з розділами:
  - Hero-секція з фоновим зображенням
  - Секція "Про нас" з фото інтер'єру
  - Фірмові страви
  - Заклик до бронювання столика
  - Найближчі події
  - Відгуки клієнтів
  - Спеціальні пропозиції
  - Заклик до контакту

- **about.html** - сторінка "Про нас" з розділами:
  - Історія ресторану
  - Філософія
  - Шеф-кухар
  - Команда
  - Відгуки
  - Заклик до бронювання

- **contact.html** - контактна сторінка з:
  - Формою зворотного зв'язку
  - Контактною інформацією
  - Картою Google Maps
  - Годинами роботи

### Функціональні сторінки
- **menu_list.html** - сторінка меню з категоріями та стравами
- **dish_detail.html** - детальна сторінка страви
- **reservation_form.html** - форма бронювання столика
- **event_list.html** - список подій
- **event_detail.html** - детальна сторінка події
- **cart.html** - кошик замовлення
- **checkout.html** - оформлення замовлення

## Стилізація

### Основні кольори
- Основний колір: #1e3a8a (темно-синій)
- Акцентний колір: #d4af37 (золотий)
- Фон: #ffffff (білий) та #f8f9fa (світло-сірий)
- Текст: #212529 (темно-сірий) та #6c757d (сірий)

### Типографіка
- Заголовки: Playfair Display (елегантний шрифт із засічками)
- Основний текст: Poppins (сучасний шрифт без засічок)

### Компоненти
- **Картки** - для відображення страв, подій, відгуків
- **Кнопки** - стилізовані в кольоровій гамі ресторану
- **Форми** - з плаваючими підписами та валідацією
- **Навігація** - фіксована навігаційна панель з логотипом
- **Слайдери** - для відображення відгуків та акцій
- **Анімації** - плавні анімації при прокрутці (fade, slide)

## Інтерактивні елементи

- **Кнопка "Назад до верху"** - з'являється при прокрутці
- **Анімація "Додати в кошик"** - візуальний ефект при додаванні страви
- **Плавна прокрутка** - при переході по якірних посиланнях
- **Модальні вікна** - для швидкого перегляду інформації
- **Валідація форм** - перевірка введених даних у реальному часі
- **Лічильник кількості** - для вибору кількості страв

## Адаптивність

- Повністю адаптивний дизайн для всіх пристроїв:
  - Десктоп (1200px+)
  - Ноутбук (992px-1199px)
  - Планшет (768px-991px)
  - Мобільний (576px-767px)
  - Малий мобільний (<576px)

- Адаптивні елементи:
  - Навігаційне меню згортається в "бургер-меню" на малих екранах
  - Сітка з карток змінює кількість колонок залежно від розміру екрану
  - Розмір шрифтів та відступів адаптується
  - Зображення масштабуються пропорційно
